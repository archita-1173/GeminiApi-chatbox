# GeminiApi-chatbox

## Description
A simple React-based chatbox that uses the Gemini API to generate AI-powered responses to user questions.

## Features
- Ask questions and get AI-generated answers using Gemini API
- Simple, clean UI

## Getting Started

### Prerequisites
- Node.js (v16 or higher recommended)
- npm or yarn

### Installation
1. Clone the repository:
   ```sh
   git clone https://github.com/archita-1173/GeminiApi-chatbox.git
   cd GeminiApi-chatbox
   ```
2. Install dependencies:
   ```sh
   npm install
   # or
   yarn install
   ```

### Running the App
```sh
npm run dev
# or
yarn dev
```

Open your browser to the local server URL shown in the terminal (usually http://localhost:5173).

## Usage
Type your question in the textarea and click "Generate Answer" to get a response from the Gemini API.
